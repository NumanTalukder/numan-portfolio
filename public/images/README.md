# Photos & project images

Drop the real image files here using the exact filenames referenced in
`src/data/content.ts`. Until a file exists, the UI shows a branded placeholder.

Expected files:

- `numan-tea.jpg` — tea-garden portrait (About section)
- `numan-forest.jpg` — forest portrait (used in later phases)
- `proj-1.jpg` … `proj-4.jpg` — project thumbnails

To change which file a section uses, edit the path in `content.ts` — never the components.
